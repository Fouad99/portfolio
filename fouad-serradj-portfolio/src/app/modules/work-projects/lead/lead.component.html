Lead

The Lead program is a web based application designed for team leaders and quality control managers for Rogers call centers. The goal of the program is to allow team leaders to enter data of their agents on a daily basis in order to keep track of their performance. The program compiles thos data and connect with other services to produce KPI numbers and analysis elements.

The Lead program is based on a decoupled architecture. Frontend code is separated from the backend code. The frontend and backend code communicate, but have their own server instances. The backend application serves data via an API in a JSON format, as part of a headless set-up. It also connects to other services (Agent presence and time tracking services) in order to produce KPI numbers and analysis elements.
