GIS 52 Biskra

A software that manages geographic information system (GIS) of the Wilaya of Biskra in Algeria. The GIS are first developped by either MapInfo or ESRI softwares, and then embedded in a GIS software that we implement. The software allows its users browse the maps of the GIS and perform research queries. It also allows CRUD operations on its database.

The GIS-52-Biskra software is based on a desktop application architecture that runs on Windows operating systems. Itself is composed of several dependent modules.

  • RDMS : MS SQL
  • Query language : SQL
  • Language :
    • Delphi Borland
    • Visual Basic Application
